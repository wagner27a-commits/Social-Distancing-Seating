# Social Distancing Seating
Seating Procedure.ipynb is a Jupyter Notebook containing SageMath code that takes the input of the size of a bar and outputs the possible bar configurations based on where the first person sits, also giving information about which starting positions allow the bar to reach maximal occupancy.
You will note that this only tests half of the starting positions. This is because if you were to have a starting position f that reaches maximal occupancy, the starting position n-f+1 will also reach maximal occupancy, as you can just essentially flip the bar around.
In order to run this code, you will need to either download SageMath or use the Sage Cell Server (https://sagecell.sagemath.org/) online.
